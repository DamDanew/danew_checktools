# Danew CheckTool - Foundation Roadmap

- Architecture et workflows
- Catalogues et profils metadata-driven
- Schémas JSON de reference
- Squelette PowerShell modulaire
- Mode simulation et recommendation baseline

## Prochaine iteration (Phase 2)

## KPI techniques
- Temps scan standard <= 4 min (WIM monte moyen)
- False negative outils critiques < 2%
- 100% mode simulation sans ecriture production
- 100% objets reportes schema-valides
