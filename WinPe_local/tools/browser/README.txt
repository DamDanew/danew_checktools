Danew WinPE browser slot

Place one portable offline browser executable here to open local HTML reports in WinPE.

Preferred names (resolution order):
- chromium.exe
- chrome.exe
- msedge.exe
- FirefoxPortable.exe

Expected path on the USB data partition:
E:\tools\browser\chromium.exe

Notes:
- Do not require internet access.
- Keep the browser self-contained in this folder.
- Test boot in WinPE before using it in SAV workflow.
- If no browser is present, reports are still generated under E:\reports and can be opened later from Windows.
